B-Banner Slider for Banner slides.

== Description ==

With B-Banner Slider you can manage all your banner stuff through admin.

Features

*   Set Banner width and height from admin section.
*   Banner auto start option.
*   Banner effects.
*   Transition delay option.
*   Thumb option.
*   PlayButton option.
*   Shuffle banner while rotating.

== Installation ==

*   Copy the B-Banner Slider plugin to your wp-content/plugins/ folder.
*   Activate the plugin in the wp-admin menu.
*   Add your first banner (http://www.example.com/wp-admin/admin.php?page=B-Banner-Slider/bbanner_slider.php)
*   Insert in your template php(`<?php if(function_exists('bbanner_slider')) { bbanner_slider(); } ?>`)
